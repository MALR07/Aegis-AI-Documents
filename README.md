# 📖 Descripción

**Aegis AI Documents** es un proyecto de automatización inteligente desarrollado como parte del curso de **RPA + Inteligencia Artificial**, cuyo objetivo es transformar un proceso tradicionalmente manual de recepción, validación y comparación de presupuestos en un flujo completamente automatizado, trazable y escalable.

La solución combina tecnologías de **Power Automate Cloud**, **Microsoft SharePoint** e **Inteligencia Artificial** para automatizar la gestión documental desde la recepción de los presupuestos hasta la generación de un informe comparativo final, proporcionando una recomendación objetiva basada en criterios previamente definidos.

El proyecto se ha diseñado siguiendo metodologías de desarrollo colaborativo similares a las utilizadas en entornos empresariales reales, aplicando control de versiones mediante Git y una organización del trabajo basada en buenas prácticas de ingeniería de software.

# 🎯 Objetivos del Proyecto

La automatización persigue mejorar significativamente la eficiencia del proceso mediante:

# 🏗️ Arquitectura General

La solución está dividida en dos procesos principales:

### Dispatcher

Responsable de la recepción de correos electrónicos, validación inicial de la documentación, organización de expedientes y preparación de los documentos para su posterior procesamiento.

### Performer

Encargado de procesar los expedientes pendientes, extraer información mediante Inteligencia Artificial, validar los datos obtenidos, comparar las ofertas recibidas, generar una puntuación objetiva y producir el informe comparativo final.

Ambos procesos trabajan de forma coordinada utilizando SharePoint como repositorio central y fuente de información compartida.
